CSCI E-12 Final Project

Welcome to my final project! My project plan was to write a simple web app that recreates the experience of drawing on a notepad or on a whiteboard (which is why everything is done in blue pen and marker colors). In keeping with that goal, I decided to try a flat design pattern that helps the user focus on the functionality of the site - drawing sketches - with as few visual distractions from the site layout and navigation as possible. Libraries and resources used:

  • Zurb Foundation 5.0
  • jQuery 2.1.3
  • Handlebars.js 3.0.1
  • Sketch.js 0.0.1
  • Colorbox.js 1.4.33
  • PHP 5.4
  • Fontspring.com - Civita Web Font
  • Iconmonstr.com - PNG & SVG Icons
